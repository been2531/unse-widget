## ✅ 완료된 작업

### 핵심 기능
- ✅ 홈화면 포켓몬 TCG 카드 스타일 — 3D 틸트, 홀로그램, 원소 이펙트
- ✅ 운세 시스템 — 5종 운세, 광고 해금 (mock)
- ✅ 가챠 뽑기 — 1회/10회, 카드 플립 애니메이션, 코인 시스템
- ✅ 가챠 확률 5단계화 — mythic(1%) 추가, 70/20/7/2/1 확률
- ✅ 합성 시스템 — 동일 카드 4~5장 → 상위 등급, 성공률 65/40/20/10%
- ✅ 광고 코인 10코인/회 (5회/일 한도)
- ✅ 무료뽑기 1회/일 로직 (freePulls.ts)

### 카드 & 캐릭터
- ✅ AI 생성 캐릭터 아트 6원소 × 3~4단계 (Pollinations.ai)
- ✅ 흰 배경 제거 (BFS flood-fill)
- ✅ 원소 이펙트 공용 유틸 분리 (elementEffects.ts)
- ✅ 한국 신화 캐릭터 58종+ — 삼족오/이무기/백호/사슴/구미호/봉황/도깨비/해태 + 사신
- ✅ Mythic 6종 추가 — 천화신/용왕/뇌신/목신/명부왕/태양신 (CARD_POOL)
- ✅ cardImageFor() — 원소+등급 → 이미지 자동 매핑 (mythic은 _3 폴백)

### 홈화면 카드 개선 (TCG 스타일)
- ✅ 카드 하단 정보 섹션 — 캐릭터명 + 원소 뱃지 + 운세 텍스트 + 희귀도 별
- ✅ 아트워크 창 내부 프레임 — Skia RoundedRect 이중 테두리 (원소색 + 흰 하이라이트)
- ✅ 배경 아트워크 창 국한 — Skia Group clip으로 그라디언트 내부 프레임에 한정
- ✅ 이펙트 Rare+ 전용 — Common은 파티클/발광 없음
- ✅ 캐릭터 팝아웃 3D 효과 — CHAR_BLEED(20%)만큼 내부 프레임 아래로 삐져나옴
- ✅ 외부 테두리 Layer 6 — 캐릭터 위에 렌더링해서 프레임이 캐릭터를 담는 효과
- ✅ 원소 선택 피커 — 미보유 원소 🔒 잠금, 저장
- ✅ 등급 선택 피커 — 5단계, 미보유 등급 🔒 잠금, 최고 보유 등급 자동 선택
- ✅ 캐릭터명 CARD_POOL 기반 — 원소+등급 조합 실제 nameKo 표시
- ✅ ScrollView 적용 — 홈화면 세로 스크롤 지원
- ✅ cardCustomization 저장 — 선택 원소·등급 AsyncStorage 영속화

### 컬렉션 & UI
- ✅ 컬렉션 화면 — 카드 도감 (미수집 흐리게), 탭 필터, 카드 탭 상세 모달

## 🔲 남은 작업 — 우선순위 순

### 🔴 즉시 필요 (네이티브 빌드)

#### 1. 광고 SDK 연동
- `react-native-google-mobile-ads` 설치
- AdMob 앱 ID → AndroidManifest.xml
- 리워드 광고 테스트 ID로 실제 시청 → 코인 지급 연결
- 현재: 3초 mock 타이머로 대체 중

#### 2. IAP 연동
- `react-native-iap` 설치
- Google Play Console에서 제품 3종 등록
  - `coins_100` (₩1,100), `coins_330` (₩2,900), `coins_1100` (₩8,900)
- 결제 후 코인 지급 로직
- 현재: 미구현 안내 Alert만 표시

### 🟡 캐릭터/콘텐츠

#### 3. 캐릭터 얼굴/손 팝아웃 개선
- 사용자 요청: "얼굴과 손이 카드 안쪽의 레이아웃 네모칸은 넘어가는 모습으로 구성해서 입체감을 주자"
- 현재: 캐릭터가 내부 프레임 **하단**으로만 CHAR_BLEED 삐져나옴
- 개선 방향: 캐릭터 이미지 상단(얼굴)도 내부 프레임 위로 더 명확히 삐져나오도록
  - 방법 A: `top` 오프셋 마이너스(-BLEED_TOP)로 캐릭터 위치 조정 (현재 overflow:hidden과 충돌 우려)
  - 방법 B: 아트워크 창 y좌표 아래로 내려서 캐릭터 상단 여유 확보

#### 4. Mythic 캐릭터 아트 생성
- `fire_4.png`, `water_4.png` 등 실제 mythic 이미지 없음
- 현재 cardImageFor()가 `_3`으로 폴백 처리 중
- Pollinations.ai로 6장 생성 필요

#### 5. ScrollView ↔ GestureDetector 충돌 확인
- 홈카드 3D 틸트(PanGesture)가 ScrollView 안에 있어 세로 스크롤과 제스처 충돌 가능
- 에뮬레이터 실행 후 직접 확인 필요
